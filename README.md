# Sales Executive Performance Dashboard

## Overview
An interactive Excel dashboard to track and analyze sales executive performance across different cities in India.

## Features
- **City-wise Analysis**: View data for Chennai, Delhi, Mumbai, Nagpur, Patna, Pune, Ranchi, and Surat
- **Multiple Dashboards**: 4 different dashboard views for comprehensive analysis
- **Performance Metrics**:
  - Top 5 Total Sales
  - Bottom 5 Total Sales
  - Target Hit % wise
  - Away From Target Hit %
- **Visual Charts**: 
  - Bar charts for sales comparison
  - Pie chart for sales distribution
  - Line chart for target achievement tracking

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Slicers
- Data Cleaning & Formulas

## How to Use
1. Download the Excel file
2. Select a city from the top menu
3. Choose a dashboard view (1-4)
4. Interact with the charts and data tables
